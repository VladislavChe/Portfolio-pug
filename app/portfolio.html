Практика JavaScript

Практика своими руками

01

Calculator

Калькулятор подсчета суммы покупки в соответствии с выбранным планом и количеством лицензий. Цена динамичная и легко заменима на другую стоимость.
02

Accordeon

При клике на пункт меню удаляем класс "active" у всех пунктов, добавив класс тому на который был клик. Высота контента считается в зависимости от количества текста.
03

CSS Generator

Получаем все input и создаем для них переменные. Берем значения input и связываем их со значениями радиуса куба. При изменении значений радиус меняется.
04

Slider

Создаем переменную равную индексу слайда. Передаем ее в функцию которая добавляет ей класс "active". Делаем связку активной точки и активного слайда.
05

Tabs

Пишем функцию, которая принимает аргумент из вне (конкретный элемент по которому кликнули), добавляя этому элементу класс "active". Связываем кнопки и контент.
06

Map

Ловим событие движения мыши на шкале увеличения карты. Приравниваем ширину и высоту карты к значениям положения мыши. Карта увеличивается без кликов мышкой.
